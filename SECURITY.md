Reporting a Vulnerability

Move78 International takes the security of our tools seriously.

If you discover a vulnerability in Shadow AI Hunter (e.g., a way for the scanner to be used for DoS amplification, or a flaw in the logic), please do NOT open a public issue.

Please report sensitive security issues via email to: contact@move78int.com 

We will acknowledge your report within 48 hours and provide a timeline for a fix.
